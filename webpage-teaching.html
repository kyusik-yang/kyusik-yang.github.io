assemblykor New R Package
An R package providing curated Korean National Assembly datasets for teaching quantitative methods in political science. A Korean politics counterpart to palmerpenguins. Ships 7 ready-to-use datasets and 6 interactive Korean-language tutorials covering the full methods sequence from tidyverse basics to network analysis.

Multimodal Embedding for Political Science Tutorial Learning by Doing
An open research notebook and methods tutorial for embedding Korean politicians' YouTube Shorts (text + audio + video) into a shared vector space using Google's Gemini Embedding 2. Documents the full pipeline from API setup to panel regression, written for political scientists without ML background. With Yongjai Yu (UC Riverside).
4 chapters 5-phase self-study curriculum 30+ curated readings
